# Web Map Previews v1

This directory stores generated browser map-preview artifacts for recovered
HectaresBC layers.

Generated by the main repository:

```text
scripts/generate_layer_preview_batch.py
```

Contents:

- `index.json`: compact index for all recovered catalog records.
- `layers/<dataset_id>/manifest.json`: per-layer preview metadata.
- `layers/<dataset_id>/preview.png`: generated RGBA preview payload, tracked by
  git-annex.
- `validation/<run_id>/generation_summary.json`: generation counts.
- `validation/<run_id>/failed_layers.csv`: non-previewable or failed layers for
  that run.

Current generation status is recorded in `index.json`. Records with
`artifact_status: not_previewable` were readable source rasters but did not
produce visible preview pixels at the tested preview size.
