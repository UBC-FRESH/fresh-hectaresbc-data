# Validation Reports

This directory contains compact reports generated by Phase 3 validation scripts.

## Current Reports

- `recovered_catalog_validation.md`: validates recovered catalog counts, IDs, joins, JSON-in-CSV fields, and expected payload-member fields.
- `representative_payload_validation.md`: validates a small representative set of source ZIPs and raster metadata readability when tooling is available.

## Regeneration

```bash
python scripts/validate_recovered_catalog.py
python scripts/inspect_representative_payloads.py
```
