# Metadata

This directory mirrors compact public metadata from the main `fresh-hectaresbc` repository so the DataLad data repository can be inspected and validated without relying on private local source paths.

Mirrored metadata includes archive inventory outputs, recovered catalog records, catalog schema notes, and validation reports. Raw HectaresBC root control files live beside the raw payload layout under `raw/hectaresbc_2022_export/`.

Deferred or excluded from this data repository:

- disposable acquisition notebooks;
- `.ipynb_checkpoints/`;
- local `tmp/` content;
- credentials and local logs;
- extracted rasters and derived products not covered by a derived-data contract.
